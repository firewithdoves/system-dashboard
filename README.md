# 💻 System Dashboard

![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)

A live system dashboard built with Python, `psutil`, and `rich`.

## Features
- 🧠 CPU usage and core count  
- 🧵 Memory usage  
- 💾 Disk usage (C:)  
- ⏱️ Uptime display  

## Run
```bash
python system_dashboard.py
